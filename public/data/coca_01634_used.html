used 
英[ju:st] 美[just]
 
adj.用过的；习惯于；二手的；旧的；v.使用（ use的过去式和过去分词）；常常，经常；
 "

PHR-MODAL
过去常常；过去曾
If something used to be done or used to be the case, it was done regularly in the past or was the case in the past.
例：People used to come and visit him every day...人们过去每天都来拜访他。
例：He used to be one of the professors at the School of Education...他曾是教育学院的一名教授。

PHR-MODAL
过去不曾（亦作did not use to 或 did not used to，尤用于口语）
If something used not to be done or used not to be the case, it was not done in the past or was not the case in the past. The forms did not use to and did not used to are also found, especially in spoken English.
例：Borrowing used not to be recommended...过去是不建议借贷的。
例：At some point kids start doing things they didn't use to do. They get more independent...从某个时候起，孩子们开始做以前不曾做的事情。他们变得更加独立。

PHRASE
习惯于
If you are used to something, you are familiar with it because you have done it or experienced it many times before.
例：I'm used to having my sleep interrupted...我习惯了睡觉时被吵醒。
例：It doesn't frighten them. They're used to it.这可吓不倒他们。他们已经习惯了。

PHRASE
开始习惯于
If you get used to something or someone, you become familiar with it or get to know them, so that you no longer feel that the thing or person is unusual or surprising.
例：This is how we do things here. You'll soon get used to it...我们这里就是这么做事的。你很快就会习惯的。
例：He took some getting used to...他过了一段时间才慢慢适应。
