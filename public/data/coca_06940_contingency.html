contingency 
英[kənˈtɪndʒənsi] 美[kənˈtɪndʒənsi]
 
n.意外事故，偶发事件；可能性，偶然性；[审计]意外开支；
 "

N-VAR
可能发生的事;不测之事;突发事件
A contingency is something that might happen in the future.
例：I need to examine all possible contingencies.我需要审视一切可能出现的情况。

ADJ
(计划或措施)未雨绸缪的，应急的
A contingency plan or measure is one that is intended to be used if a possible situation actually occurs.
例：We have contingency plans.我们有应急方案。
