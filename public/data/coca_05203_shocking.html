shocking 
英[ˈʃɒkɪŋ] 美[ˈʃɑkɪŋ]
 
adj.令人震惊的，可怕的；触目惊心的；〈口〉非常粗陋的；adv.〈口〉极度[说不出地]（坏）；v.使…震惊( shock的现在分词)；
 "

ADJ-GRADED
糟透了的;极坏的
You can say that something is shocking if you think that it is very bad.
例：The media coverage was shocking...当时媒体的报道糟透了。
例：I must have been in a shocking state last night.昨晚我的情况一定很糟糕。

ADJ-GRADED
令人震惊的;令人愤慨的;不道德的
You can say that something is shocking if you think that it is morally wrong.
例：It is shocking that nothing was said...当时竟然什么都没说,真让人愤慨。
例：This was a shocking invasion of privacy.这是对隐私的公然侵犯。
