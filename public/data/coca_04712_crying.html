crying 
英[ˈkraɪɪŋ] 美[ˈkraɪɪŋ]
 
adj.（尤指坏事）极糟糕的；（表示愤怒或失望）哎呀；（用于强调）极其糟糕；迫切的需要；v.哭( cry的现在分词 )；喊，叫；叫卖，大声报道，哭着说；鸣；
 "

PHRASE
迫切的需要
If you say that there is a crying need for something, you mean that there is a very great need for it.
例：There is a crying need for more magistrates from the ethnic minority communities.亟须更多来自少数民族群体的治安官。

PHRASE
让人不安的事;使人烦恼的事;奇耻大辱
You can say that something is a crying shame if you are annoyed and upset about it.
例：It's a crying shame that police have to put up with these mindless attacks.令人气愤的是，警察不得不忍受这些无知的批评。
