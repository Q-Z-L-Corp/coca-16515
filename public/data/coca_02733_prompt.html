prompt 
英[prɒmpt] 美[prɑmpt]
 
adj.敏捷的；迅速的；立刻的；准时的；v.促使；导致；鼓励；提示；n.激励；提示符；提词；提示；adv.准时地；
 "

VERB
促使；推动；引起
To prompt someone to do something means to make them decide to do it.
例：Japan's recession has prompted consumers to cut back on buying cars...日本经济的不景气使得消费者在购买车辆上减少了开支。
例：The need for villagers to control their own destinies has prompted a new plan.村民们想要掌握自己的命运，这促成了一项新的计划。

VERB
鼓励，提示（说话者）；给（演员）提词
If you prompt someone when they stop speaking, you encourage or help them to continue. If you prompt an actor, you tell them what their next line is when they have forgotten what comes next.
例：'Go on,' the therapist prompted him...“接着说，”医生鼓励他道。
例：How exactly did he prompt her, Mr Markham?确切地说，他是怎样提示她的，马卡姆先生？

ADJ-GRADED
迅速的；即刻的；立即的；及时的
A prompt action is done without any delay.
例：It is not too late, but prompt action is needed.现在还不算晚，但是需要立即采取行动。
例：...an inflammation of the eyeball which needs prompt treatment.需要立即治疗的眼球炎症

ADJ-GRADED
敏捷的；迅速的；及时的
If you are prompt to do something, you do it without delay or you are not late.
例：You have been so prompt in carrying out all these commissions...你执行所有这些命令可真够迅速的。
例：We didn't worry because they were always so prompt with their rental payment.我们不担心，因为他们交租金总是非常及时。
