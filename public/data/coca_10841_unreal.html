unreal 
英[ˌʌnˈrɪəl] 美[ˌʌnˈriəl]
 
adj.不真实的，假的；幻想的，虚构的；非现实的；虚幻；
 "

ADJ-GRADED
不真实的；怪异的；难以相信的
If you say that a situation is unreal, you mean that it is so strange that you find it difficult to believe it is happening.
例：It was unreal. Like some crazy childhood nightmare...这不是真的，就像儿时的某个荒唐的噩梦一样。
例：It felt so unreal to be talking about our son like this.真不敢相信我们会这样谈论我们的儿子。

ADJ-GRADED
不可能的；虚构的
If you use unreal to describe something, you are critical of it because you think that it is not like, or not related to, things you expect to find in the real world.
例：...unreal financial targets...不切实际的理财目标
例：Almost all fictional detectives are unreal.几乎所有小说中的侦探都是虚构的。

ADJ-GRADED
极好的
Some people use unreal to indicate that they like something very much.
例：Oh, I can't explain it. It's just unreal. Everybody is so happy.啊，我都没法形容。就是太棒了，每个人都很高兴。
