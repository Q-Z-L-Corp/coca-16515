filing 
英[ˈfaɪlɪŋ] 美[ˈfaɪlɪŋ]
 
n.整理成档案，文件归档，锉；v.提出（离婚诉讼或其他讼案）( file的现在分词 )；把…归档；用锉锉；（法院或其它行政机关）定案；
 "

N-COUNT
文件箱;文件夹
A file is a box or a folded piece of heavy paper or plastic in which letters or documents are kept.
例：He sat behind a table on which were half a dozen files.他坐在一张放有6个文件夹的桌子后面。
例：...a file of insurance papers.一夹子保险文件

N-COUNT
档案;卷宗
A file is a collection of information about a particular person or thing.
例：There was stuff in that file that was private between me and Dr Denny...那份档案的有些内容是只应我和丹尼医生知道的。
例：We already have files on people's tax details, mortgages and poll tax...我们已经建立了人们纳税详情、抵押贷款以及人头税的档案。

VERB
把(文件等）归档
If you file a document, you put it in the correct file.
例：A secretary can file papers as efficiently as a floppy disk can store them...秘书归档文件时可以像软盘存储一样高效。
例：They are all filed alphabetically under author.这些都是按照作者姓名的字母顺序归档的。

N-COUNT
(计算机的)文件
In computing, a file is a set of related data that has its own name.

VERB
提起(诉讼);提出(投诉、请求等)
If you file a formal or legal accusation, complaint, or request, you make it officially.
例：A number of them have filed formal complaints against the police...他们中的许多人都对警方提出了正式控告。
例：I filed for divorce on the grounds of adultery a few months later.几个月后我以通奸为由提起了离婚诉讼。

VERB
发送，送交(新闻报道等)
When someone files a report or a news story, they send or give it to their employer.
例：Catherine Bond filed that report for the BBC from Nairobi...凯瑟琳·邦德从内罗毕向英国广播公司发送了那篇报道。
例：He had to rush back to the office and file a housing story before the secretaries went home.他不得不冲回办公室，赶在秘书们回家之前提交一篇有关住房的新闻报道。

VERB
排成纵列行进
When a group of people files somewhere, they walk one behind the other in a line.
例：She paused as the group of children filed out of the house...那群孩子从房子里鱼贯而出时，她停了下来。
例：Slowly, people filed into the room and sat down.人们排成一列缓慢地依次走进房间，并坐了下来。

N-COUNT
锉刀
A file is a hand tool which is used for rubbing hard objects to make them smooth, shape them, or cut through them.

VERB
(用锉刀)锉
If you file an object, you smooth it, shape it, or cut it with a file.
例：Manicurists are skilled at shaping and filing nails.美甲师修锉指甲的手艺娴熟。

PHRASE
记录(或保存)在档的
Something that is on file or on someone's files is recorded or kept in a file or in a collection of information.
例：His fingerprints were on file in Washington...他的指纹已在华盛顿记录在档。
例：We'll keep your details on file...我们会把有关你的详细信息记录在档。

PHRASE
成纵队;成单列
A group of people who are walking or standing in single file are in a line, one behind the other.
例：We were walking in single file to the lake.我们正排成一列朝湖边走去。

PHRASAL VERB
把(文件等)归档
If you file away a document, you put it in the correct file.
例：When at home, most evenings would be spent reading, cutting up and filing away his photographs...晚上在家时，经常都是在看书或剪切和归类他的照片。
例：The details are filed away by his three secretaries for future reference.他的3个秘书已经把详细情况归档备查。
