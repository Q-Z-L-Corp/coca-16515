useful 
英[ˈju:sfl] 美[ˈjusfəl]
 
adj.有用的，有益的，有帮助的；令人满意的；
 "

ADJ-GRADED
有用的；可用的；有助益的
If something is useful, you can use it to do something or to help you in some way.
例：The slow cooker is very useful for people who go out all day...慢炖锅对于整天都不在家的人来说很有用。
例：Hypnotherapy can be useful in helping you give up smoking...催眠疗法可能有助于你戒烟。

PHRASE
有用；派得上用场
If an object or skill comes in useful, it can help you achieve something in a particular situation.
例：The accommodation is some distance from the clubhouse, so a hire car comes in useful.住处离会所有一段距离，所以租一辆车会很有用。
