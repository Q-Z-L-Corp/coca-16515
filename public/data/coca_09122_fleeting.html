ADJ-GRADED
短暂的;飞逝的
Fleeting is used to describe something which lasts only for a very short time.
例：The girls caught only a fleeting glimpse of the driver...女孩们只匆匆瞥了一眼司机。
例：She wondered for a fleeting moment if he would put his arm around her.他会不会搂住自己，这个念头在她脑子里一闪而过。
