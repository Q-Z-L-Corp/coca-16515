ADJ
(颜色、款式)相同的，一致的，相配的，相称的
Matching is used to describe things which are of the same colour or design.
例：She bought more fabric in the same design so she could make matching curtains and cushions.她又买了些图案相同的布，好做些配套的窗帘和垫子。
例：...a coat and a matching handbag.大衣和与之相配的手提包
