dynamite 
英[ˈdaɪnəmaɪt] 美[ˈdaɪnəˌmaɪt]
 
n.炸药；具有爆炸性的事物，有潜在危险的人；引起轰动的人(或物；vt.破坏，炸毁；装炸药；adj.极好的；
 "

N-UNCOUNT
甘油炸药
Dynamite is a type of explosive that contains nitroglycerin.
例：Fifty yards of track was blown up with dynamite.50 码铁轨被炸毁。

VERB
炸毁
If someone dynamites something, they blow it up by using dynamite.
例：The rebels dynamited power lines...叛乱分子炸毁了输电线。
例：The slum is to be dynamited.贫民窟要被炸掉了。

N-UNCOUNT
爆炸性消息;引起轰动的事物
If you describe a piece of information as dynamite, you think that people will react strongly to it.
例：Her diaries are political dynamite.她的日记是政治“炸药”。

N-UNCOUNT
令人振奋的人（或事物）；令人激动的人（或事物）
If you describe someone or something as dynamite, you think that they are exciting.
例：Carmen was pure dynamite...卡门充满激情。
例：The first kiss is dynamite.初吻令人心潮澎湃。
