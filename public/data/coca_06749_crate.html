crate 
英[kreɪt] 美[kret]
 
n.板条箱，柳条箱；装货箱；一箱（之量）；（俚）破旧的汽车（飞机）；vt.<美>（用篮筐或板条箱）把…装入箱中；
 "

N-COUNT
(运输或储藏物品的)板条箱，大木箱
A crate is a large box used for transporting or storing things.
例：...a pile of wooden crates...一堆大木箱
例：A crane was already unloading crates and pallets.起重机已经开始卸载木箱和托盘了。

VERB
把…装进木箱(或板条箱)
If something is crated, it is packed in a crate so that it can be transported or stored somewhere safely.
例：The much repaired plane was crated for the return journey.经过大修的刨子被装入木箱运回去。

N-COUNT
(运送瓶子用的)塑料分格箱，金属分隔箱
A crate is a plastic or wire box divided into sections which is used for carrying bottles.
