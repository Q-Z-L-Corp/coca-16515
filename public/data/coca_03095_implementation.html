VERB
履行;实施
If you implement something such as a plan, you ensure that what has been planned is done.
例：The government promised to implement a new system to control financial loan institutions...政府许诺实施新的制度来控制金融贷款机构。
例：The report sets out strict inspection procedures to ensure that the recommendations are properly implemented.报告规定了严格的检查程序，以确保建议得到切实执行。

N-COUNT
工具;器具;用具
An implement is a tool or other piece of equipment.
例：...knives and other useful implements.刀子和其他有用工具
